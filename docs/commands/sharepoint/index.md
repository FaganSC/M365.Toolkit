# SharePoint commands

| Command | Description |
| --- | --- |
| [Connect-SP365](Connect-SP365.md) | Connects interactively to a saved SharePoint Online tenant. |
| [Clear-SP365List](Clear-SP365List.md) | Deletes or recycles every item in a SharePoint list. |
| [Copy-SP365Nav](Copy-SP365Nav.md) | Copies navigation nodes between locations or sites. |
| [Move-SP365Nav](Move-SP365Nav.md) | Moves navigation nodes between locations or sites. |
| [Restore-SP365ListItems](Restore-SP365ListItems.md) | Restores a list's items from the SharePoint recycle bin. |
